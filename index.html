My Projects

Project Title : Vehicle Parking Management System

Description : This project was conceived to optimize vehicle parking management, facilitating efficient entry recording and charge computation for the duration of stay.

Key Features :

  • Vehicle Entry Tracking: Captures vehicle details including entry time, duration of stay, and departure.
  • Automated Charge Calculation: Calculates charges based on the duration of parking.
  • Generation of Payment Receipts: Provides detailed slips displaying incurred charges for each vehicle.

Technologies Used:

  • Frontend Implemented using HTML, CSS, JavaScript, and Bootstrap 5 for a responsive and intuitive user interface.
  • Backend : Employed Node.js along with the Express.js framework to manage server-side operations and develop RESTful APIs.
  • Database: Utilized MongoDB to efficiently store and manage parking records, enabling seamless CRUD operations.

Achievements:

  • Developed a user-friendly interface allowing smooth data entry and retrieval.
  • Successfully implemented backend functionalities, handling data operations and calculations accurately.
  • Uploaded the project to GitHub for public access and collaboration. Vehicle Parking
