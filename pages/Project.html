Vehicle Parking Management System

Developed a comprehensive vehicle parking management system to streamline entry recording and charge computation for parking duration. Utilized a frontend stack comprising HTML, CSS, JavaScript, and Bootstrap 5 to craft a responsive, user-friendly interface. The backend, powered by Node.js and Express.js, efficiently handled server-side operations and facilitated RESTful API development. MongoDB served as the database, enabling seamless CRUD operations and efficient data storage. Notable achievements include the successful implementation of entry tracking, accurate charge calculations, and the generation of detailed payment receipts. This project is publicly available on GitHub: Vehicle Parking Management System.

Key Features:

  • Vehicle Entry Tracking: Captures vehicle details including entry time, duration of stay, and departure.
  • Automated Charge Calculation: Calculates charges based on the duration of parking.
  • Generation of Payment Receipts: Provides detailed slips displaying incurred charges for each vehicle.

Technologies Used

  • Frontend: Implemented using HTML, CSS, JavaScript, and Bootstrap 5 for a responsive and intuitive user interface.
  • Backend: Employed Node.js along with the Express.js framework to manage server-side operations and develop RESTful APIs.
  • Database: Utilized MongoDB to efficiently store and manage parking records, enabling seamless CRUD operations.
